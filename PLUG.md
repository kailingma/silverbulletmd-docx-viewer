# DOCX Viewer

Renders `.docx`, `.docm`, and `.dotx` files directly in SilverBullet using [docx-preview](https://github.com/VolodymyrBaydalka/docxjs). The editor is replaced with a paginated document view while the titlebar remains visible.

Requires network access to jsDelivr on first open. Legacy binary `.doc` files are not supported.
